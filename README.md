## Title of the Project

Expense Tracker : Intelligent Personal Finance Management Application

The Expense Tracker application integrates smart expense recording, real-time budget monitoring, and analytical insights to help users track, categorize, and optimize their daily spending efficiently.

## About:

The Expense Tracker application helps individuals manage their personal finances by recording daily expenses, analyzing spending patterns, and monitoring budgets in real time. The system uses data analytics to generate insightful reports and visualizations, enabling users to identify overspending and improve financial planning. Built with a Flask backend and a React frontend, the application delivers a user-friendly interface with features such as expense categorization, monthly summaries, and actionable insights like “Budget exceeded” or “Savings goal achieved.”

## Features

Automated expense categorization using rule-based logic and machine learning for accurate spending classification.

Time-series spending analysis to track daily, monthly, and yearly expense trends for better financial planning.

Real-time expense logging with historical transaction charts and category-wise breakdowns.

Custom budget allocation sliders for categories such as food, travel, utilities, and entertainment.

Responsive web interface with interactive dashboards, visual analytics, and savings insights.

## Requirements

Hardware: Intel/AMD dual-core processor or higher,4 GB minimum (8 GB recommended),100 GB free disk space,Stable internet connection.
Software:Windows / Linux / macOS, Android Studio,Java ,Gradle, Git
Libraries:Android SDK, Material Design Components,Room Persistence Library / SQLite,MPAndroidChart,Firebase Authentication (optional),Firebase Analytics

## Results and Impact

Enables accurate tracking of daily, weekly, and monthly expenses with error margin below 5% in categorization.

Generates visual reports and summaries that improve financial awareness and planning efficiency.

Enhances decision-making with personalized alerts for budget limits and savings goals.
